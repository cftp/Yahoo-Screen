== Description ==
Embed Yahoo Screen videos oembed style

== Installation ==
1. Upload to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
1. Place a Yahoo video url in post content and save

== Frequently Asked Questions ==
= Does this handle https =
Yes, it handles http and https urls.

= What about the french/german/etc sites? =
These are handled too
